# Redecta Documentation

Redecta is a privacy-first security layer that protects sensitive data before it reaches AI systems.

## Overview

Redecta helps reduce accidental data exposure in AI workflows by detecting and masking sensitive information before it is sent to AI platforms.

## What Redecta Does

Redecta applies real-time protection at the source.

It is designed to help protect:

- Personal data
- Financial information
- Credentials and secrets
- Tokens and session data
- Sensitive business input

## Security Model

Redecta is built around a privacy-first architecture.

- Local-first execution
- Zero-knowledge design
- No raw sensitive data storage
- No server-side processing of protected input

## Product Position

Redecta is not an AI assistant.

It is a security control layer between users and AI systems.

## Trust & Compliance

Redecta is built with modern privacy and security expectations in mind.

- Designed with GDPR and KVKK principles in mind
- Minimizes exposure risk by design
- Reduces unnecessary handling of sensitive input
- Built to support enterprise security and compliance requirements
